# Computer Science & Mathematics

A collection of projects exploring topics in computer science and mathematics.

## Projects

| Project | Topic | Description |
|---------|-------|-------------|
| [Fourier Transforms](math/fourier-transforms/) | Signal Processing | DFT from scratch, FFT comparison, signal decomposition, frequency filtering, and 2-D image FFT. Includes Python (Jupyter notebook), C (AVX SIMD), and C# (`System.Runtime.Intrinsics`) implementations with benchmarks |
| [GPU Parallel Demo](computational/gpu-parallel-demo/) | GPU Computing | Parallel vector addition (16M elements) — CPU serial vs GPU parallel with timing comparison. Three implementations: CUDA, OpenCL, and OpenMP target offloading |

## Structure

- **algorithms/** — Algorithm implementations and analysis
- **data-structures/** — Data structure implementations
- **math/** — Mathematical concepts, proofs, and computations
- **cryptography/** — Cryptographic algorithms and protocols
- **computational/** — Numerical methods and computational mathematics
